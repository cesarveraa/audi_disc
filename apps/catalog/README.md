# Audi Disc Catalogo

Catalogo publico para clientes, separado del panel administrativo.

## Variables

```bash
VITE_API_BASE_URL=https://tu-api.vercel.app/api/v1
VITE_SITE_URL=https://audidisc.com
VITE_WHATSAPP_PHONE=59170000000
VITE_BUSINESS_STREET_ADDRESS=Calle Junin, Zona Central
VITE_BUSINESS_LATITUDE=-19.043
VITE_BUSINESS_LONGITUDE=-65.259
```

`VITE_WHATSAPP_PHONE` es opcional. Si no esta definido, el boton abre WhatsApp con el mensaje prellenado sin fijar un numero.
`VITE_SITE_URL` se usa para canonical, Open Graph, sitemap y robots.txt.
